Below is a summary of the submission package for EBA5007 Capstone Project:

Dataset Source: NHANES 2013-2014 and 2015-2016
Due to large file size, please download the dataset from https://wwwn.cdc.gov/nchs/nhanes/Default.aspx

Final Report: FinalReport_KhooMaySze_A0198537L.pdf

Presentation Slides: Finalppt_KhooMaySze_A0198537L.pptx

Learning Journey Report: LearningJourney_KhooMaySze_A0198537L.docx

R Codes:
	(1) 01 NHANES Dataset.R (Data extraction and pre-processing for trainsets) 
	(2) 02 NHANES_without.R (Feature selection for dataset excluding variables directly linked to diabetes)
	(3) 03 trainset_FS.R (Feature selection for final dataset)
	(4) 04 testset.R (Data pre-processing for the testsets)
	(5) 05 modelling DIQ010A.R (Binary classification - modelling, evaluation, sensitivity analysis and robustness tests)
	(6) 06 modelling DIQ010B.R (Multi-class classification - modelling and evaluation)
	(7) 07 B1 AUC ROC PR.R (Multi-class classification - ROC-AUC and PR-AUC evaluation)
	(8) 08 modelling B2 SA.R (Multi-class classification - sensitivity analysis and robsutness tests)

Phase 1 Report: InterimReport_KhooMaySze_A0198537L.pdf
Phase 1 Presentation Slides: updatedPredicting Type 2 Diabetes in Asian Americans_KhooMaySze_A0198537L.pptx
